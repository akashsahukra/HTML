HTML satnds for Hyper Text Markup Language.It is used to create web pages.It describes the web page through elements and attributes
The structure of HTML is maily two parts head and body:
Head:
It has only contents which are not seen on the page It can have links to designing pages or java script files and also some links to frameworks,the main part is that the title of page is defined inside the head
Some of the tags in Head:

  • title
  • link
Body:
The entire web page whivh we see on the browser is written inside body tag like headers, navigations, footers,tables,links, images, videos, audios,sections,divisions etc.
Some of the main tags in body:
  • header
  • section
  • footer
  • div
  • anchor
  • image
And also, tags have some attributes which defines size, shape ,height, width etc.
HTML is the most and important mark-up language for everyone who wants to learn web development.It has huge community and been in development for almost 25 years and is still been though.
